Data Structures and Algorithms Visualization

Full-stack web application to visualize search and sort algorithms
D3, React.js, Typescript, Bootstrap 4, CSS
Launch
×

About

Visualize Linear Search, Binary Search, Bubble Sort, Selection Sort, Merge Sort, and Insertion Sort. Random numbers in an array are graphed as a bar chart. The values of the array are displayed and updated in an HTML table below the bar chart. They are also accessible through a tooltip that displays the value when hovered over. The Javascript Execution times are logged in Bootstrap 4 cards to allow comparison of different algorithm runtimes and for further information on the Big-O.

Methodology

  • Chose to create an application based on algorithms and data structures to solidify understanding of Big-O notation and the design of algorithms
  • D3 Library for data visualization to both challenge myself with learning a new framework and because D3 is the ideal solution for data visualization problems
  • Bootstrap 4 for increasing my experience working with it and to shift focus towards spending more time on the data visualization and underlying algorithms
  • Site is hosted on Heroku because the application server-side logic is very simple

Robynhood

Full-stack web application for managing an investment portfolio by buying and selling stocks based on their actual prices
Flask, Python, SQL Alchemy, MySQL, Jinja, Bootstrap 4, CSS
Launch
×

About

The main features are login, sign-up, stock price API, and portfolio, bought, sold, and history tables in a SQL database (create, read, and update).

Methodology

  • Database designed in SQL to leverage tables and relational queries
  • Rewrote the entire application to use Flask SQL Alchemy to learn SQL in python and about object-relational mappers
  • Used Flask as a server-side framework to gain exposure to Python programming and a framework other than express.js
  • Used jinja templating to gain exposure to frameworks other than embedded javascript templates (EJS)
  • Password hashing using Werkzeug, a flask plugin
  • Continuous integration and continuous deployment (CI/CD) with Travis CI to gain exposure to a CI/CD tool other than CircleCI
  • Developed minimum viable product (MVP) of application with SQLite locally to test it and deployed with MySQL on RDS instance for production
  • Site is hosted on Amazon Web Services (AWS) on an Elastic Beanstalk instance. Chose AWS to gain skills and knowledge about this infrastructure as a service (IaaS). Chose Elastic Beanstalk since it's most similar to Heroku and it would ease me into the complexity of AWS.

YelpCamp

Full-stack web application for uploading and sharing information about campgrounds
Node.js, Express.js, Javascript, EJS, Bootstrap, CSS
Launch
×

About

First full-stack web application. Fundamentally, it's Yelp for campsites. The main features are login, sign-up, Admin role, Google Maps API, and Create, Edit, Update, Delete (CRUD) for campgrounds, comments, and reviews. Create routes have authentication. Edit, Update, and Delete routes have authentication and authorization. The basic foundation was built in the capstone project from the Udemy course The Web Developer Bootcamp.

Methodology

  • GoormIDE to gain exposure to cloud-based IDE
  • Bootstrap 4 as the CSS framework to learn Bootstrap for the first time
  • Express.js as the Node.js application framework for its basic functionality. Some companies still use Express.js so it was important to implement it.
  • PassportJs for the authentication and authorization
  • NoSQL database for the flexibility compared to a SQL database, MongoDB in particular because of its prevalence in the industry.
  • Git for version control to learn version control
  • Site is hosted on Heroku to learn how to deploy applications
  • Database is hosted on MongoDB Atlas for production
